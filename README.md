###  Featured Projects (AgroTech & Data Engineering)

#### 🗺️ Soy Crop Tour 2026
Full-scale infrastructure for real-time harvest monitoring.
* **Tech Stack:** QGIS, QField, Python, and Streamlit.
* **Impact:** Digitized field sample collection with automated geospatial data integration and high-performance dashboard visualization.

#### ⛈️ Climatology Automation (Brazil & Argentina)
Data pipeline for monitoring precipitation and weather models (GFS/ECMWF).
* **Tech Stack:** Python (XWeather API), Task Automation, and SharePoint Integration.
* **Impact:** Automated weather map generation for trading desks, providing systematic updates consumed directly via Power BI.

#### 🚢 "Sailed" Export Pipeline (Argentina)
Data architecture for tracking soybean and corn export flows.
* **Tech Stack:** Python, SQL Server, Power BI.
* **Impact:** Normalized and processed shipping data (Line-up), enabling precise analysis of logistical competitiveness and market share.

#### 📊 PnL & Trading Dashboard
Visualization system for Mark-to-Market (MTM) and Year-over-Year (YoY) variations for soy and soymeal.
* **Tech Stack:** Streamlit, Firebase (Firestore), Python.
* **Impact:** Centralized critical financial metrics for risk management and trading desk performance.
