### 🚀 Projetos em Destaque (AgroTech & Data Engineering)

#### 🗺️ Crop Tour Soja 2026
Desenvolvimento de uma infraestrutura completa para monitoramento de safra em tempo real.
* **Tech Stack:** QGIS, QField, Python e Streamlit.
* **Impacto:** Digitalização da coleta de amostras em campo, com integração automatizada de dados geoespaciais e visualização em dashboards de alta performance.

#### ⛈️ Automação Climatológica (Brasil & Argentina)
Pipeline de dados para monitoramento de precipitação e modelos meteorológicos (GFS/ECM).
* **Tech Stack:** Python (XWeather API), Automação de scripts e SharePoint Integration.
* **Impacto:** Geração automática de mapas de chuva para suporte às mesas de trading, com atualização sistemática e consumo via Power BI.

#### 🚢 Pipeline de Exportação "Sailed" Argentina
Arquitetura de dados para rastreamento de fluxos de exportação de soja e milho.
* **Tech Stack:** Python, SQL Server, Power BI.
* **Impacto:** Normalização e processamento de dados de embarques, permitindo análise precisa do Line-up e competitividade logística.

#### 📊 PnL & Trading Dashboard
Sistema de visualização de Mark-to-Market (MTM) e variações YoY para soja e farelo.
* **Tech Stack:** Streamlit, Firebase (Firestore), Python.
* **Impacto:** Centralização de métricas financeiras críticas para a gestão de risco e performance de mesa.
